## Rangkuman Belajar dan Catatan Belajar 

Mengeluarkan pesan ke halaman
```php
<?php
	echo "Hello world";
?>
```
Deklarasi variable
```php
	$a = "rabih";
	$b = 20;
```
Membuat fungsi
```php
	function greet() {
		return "Selamat pagi";
	}
```
Membuat indexed array dan mengaksesnya
```php
	$arr = array("merah", "jingga", "kuning", "biru");
	echo $arr[0]; // merah
	echo $arr[1]; // jingga
```
Membuat associative array
```php
	$arr = array("merah" => 4, "jingga" => 1, "kuning" => 40);
```
